Weather Dashboard

Weather Dashboard

Developers are often tasked with retrieving data from another application's API and using it in the context of their own. Third-party APIs allow developers to access their data and functionality by making requests with specific parameters to a URL. Your challenge is to build a weather dashboard that will run in the browser and feature dynamically updated HTML and CSS. This is an interactive weather dashboard website that allows users to search for current weather and 5-day forecast in their favorite cities and have it saved the next time they come back.

Technologies used:

  • HTML,CSS
  • Javascript
  • Jquery
  • Local Storage
  • OpenWeather API
  • Hover.CSS
  • Bootstrap
  • Github for Deployment
